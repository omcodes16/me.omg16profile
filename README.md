- **✅ High Accuracy Detection** - Precise sensor calibration for reliable performance

### 🔐 **Smart Door Lock System** - **Embedded Security**
A **robust Arduino-based security solution** with **password authentication** and **persistent memory** for **controlled access** management.

**💾 Tech Stack**: 
- **Microcontroller**: **Arduino ATmega328P**
- **Input Device**: **4x4 Matrix Keypad**
- **Display**: **I2C LCD (16x2)**
- **Storage**: **EEPROM** (persistent memory)
- **Lock Mechanism**: **Solenoid Lock Actuator**
- **Programming**: **C++**

**⭐ Key Features**:
- **✅ Multi-Digit Password Authentication** - Secure access with custom PIN codes
- **✅ EEPROM-Based Persistence** - Password survives power loss
- **✅ Real-Time LCD Feedback** - Status display for user guidance
- **✅ Robust Debouncing Logic** - Eliminates input errors and false entries
- **✅ Hardware-Software Integration** - Demonstrates complete embedded system design
